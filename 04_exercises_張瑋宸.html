介紹¶

本次分析打算以data-driven的方式，收集到在時間點(2021/10/24 17:00左右)台灣地區的youtube熱門影片前50名資料。藉由分析其中的內容、類別及各種統計數據，再從結果歸納出熱門影片可能有什麼特性。 而在分析所選用的變數中，個人比較有興趣的為以下:

  1. 影片點閱率、讚數、留言數等等。以這些基本統計數據和熱門影片的排行做相關，看其中的關聯性。
  2. 影片時長(duration)。猜想是否影片長度比較短的影片，是否會因為較容易看完而有比較多的點閱率，因此上熱門。

以下會先從研究方法收集到的資料開始介紹，接下來從上述三方面逐個分析，最後再做個總結與討論研究的可能改進之處。

#Transform "iso 8601 string" into seconds(int).

import datetime

def get_isosplit(s, split):
    if split in s:
        n, s = s.split(split)
    else:
        n = 0
    return n, s

def parse_isoduration(s):
        
    # Remove prefix
    s = s.split('P')[-1]
    
    # Step through letter dividers
    days, s = get_isosplit(s, 'D')
    _, s = get_isosplit(s, 'T')
    hours, s = get_isosplit(s, 'H')
    minutes, s = get_isosplit(s, 'M')
    seconds, s = get_isosplit(s, 'S')

    # Convert all to seconds
    dt = datetime.timedelta(days=int(days), hours=int(hours), minutes=int(minutes), seconds=int(seconds))
    return int(dt.total_seconds())

討論¶

本次分析只挑選單一時間點的50部熱門影片，樣本數和代表性本身都有些不足。只有抓取熱門影片，並未去找尋其他「非熱門」的影片做為比較，讓本次討論比較像是「分析熱門影片的影片特性」，而非「分析什麼影片才會上熱門」。只選取單一時間點做分析，在上述影片類型的分析上，有可能被當下事件所影響，造成推論的偏誤，如:因為發生社會重大事件使得熱門影片的「新聞」類型上升。

未來可以嘗試去抓取不同時間點的熱門影片排行納入分析；此外也能夠去挑選非熱門的影片和熱門影片作比較，在歸納資訊上或許會有比較好的結果。
